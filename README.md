# Firefox extension with a few vi(m) keybindings.

* [Install](http://j.mp/hnuo2O)
* [Source on Github](http://j.mp/hKMV6t)

## Supported Commands

* h : scroll left
* j : scroll down
* k : scroll up
* l : scroll right
* g : go to the top of the page
* G (shift-g) : go to the bottom of the page
